# Gameplay (v3.2)

## Adventure mode

1. Main menu → New Game / Continue or Load Save
2. Choose Bulbasaur / Charmander / Squirtle
3. Explore the overworld (town, routes, tall grass, pond shores)
4. **E** or **H**: Center heal / Mart shop first when on service tiles; else talk to **signs/NPCs** you stand on or strictly face
5. Tall grass → wild battles (levels scale with lead); types, STAB, PP, crits, speed order
6. Bag: catch (balls + %) / potion; Switch; Run (wild)
7. Win for XP + money; catch fills party (max 6) — full party still marks Pokédex/counters
8. **P** party; **Esc** pause (Save / Party / Pokédex / Title / Quit)
9. Lead Pokémon (first conscious) follows you as a small overworld sprite

## Controls

| Key | Action |
|-----|--------|
| Arrows / WASD | Move / menus |
| Enter / Space | Confirm |
| E / H | Interact (heal/shop/talk) |
| P | Party |
| Esc | Pause / back |
| Battle 1–4 | Quick command / move select |

## Tips

- Water beats Fire; Electric fails vs Ground; SE/NVE shown in Fight menu.
- Fainted party → heal at Center (red door/floor) before grass.
- Route signs list wild species and level bands.
- Save: autosave on milestones; Pause → Save. Path: app-data `pokemon-text-game/save_v3.json`.
